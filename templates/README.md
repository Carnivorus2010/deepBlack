# templates

Part of the deepBlack operating environment project.

This directory is reserved for reusable project templates that are not active configuration
by themselves.

Current active configuration templates live under:

    config/

Current generated output lives under:

    generated/

Examples of generated output include:

    generated/dwl/design_tokens.h
    generated/nvim/
    generated/yazi/theme.toml

Session/login source templates currently live under:

    config/greetd/
    config/wayland-sessions/

Do not place active system configuration here unless the project intentionally
moves that template into this directory.
